# 🏭 POF Management System

A secure multi-role management system built in **C++** with **Qt** and **OpenCV**, featuring dual authentication via password hashing and face recognition.

## 🚀 Features

### 🔐 Dual Authentication System
- Password hashing for secure login
- **Face Recognition** via OpenCV for biometric authentication

### 👥 Multi-Role Access
- Different access levels for different roles (Admin, Manager, Employee)

### 👨‍💼 Employee Records
- Add, update, remove, and search employee profiles
- Complete employee data management

### 📦 Inventory Tracking
- Real-time inventory monitoring
- Stock level management and alerts

### 🏭 Production Logging
- Log and track production activities
- Production history and reporting

### 🔧 Equipment Maintenance Scheduling
- Schedule and track equipment maintenance
- Maintenance history and upcoming service alerts

### 💾 Persistent File-Based Storage
- All data saved across sessions via file handling

## ⚙️ How to Run

### Prerequisites
- Qt Creator (6.x or later)
- C++ compiler (MinGW 64-bit)
- OpenCV installed and configured with Qt

### Steps
1. Clone the repository
```bash
   git clone https://github.com/Wajih-Gakhar/pof-management-system.git
```
2. Open `CMakeLists.txt` in **Qt Creator**
3. Make sure **OpenCV** is linked in your CMakeLists.txt
4. Select the **Desktop Qt 6.9.0 MinGW 64-bit** kit
5. Build & Run ▶️
